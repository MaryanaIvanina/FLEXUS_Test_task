# FLEXUS Test Task
Автор: Мар'яна Іваніна

Гравець керує персонажем від третьої особи. На сцені, поруч із персонажем, розташовані два автомобілі, у які він може сідати. При наближенні до автомобіля гравець має можливість сісти в нього, після чого керування переключається з персонажа на автомобіль.
Під час керування автомобілем гравець може вийти з нього та знову повернутися до управління персонажем.

Персонаж гравця має анімації для станів idle, бігу та спринту. Автомобіль є фізичним об’єктом з реалістичною поведінкою коліс та підвіски під час руху по поверхні землі.

## Технічний стек та Архітектура
- Engine: Unity 6 (URP)
- DI Container: Zenject (Extenject)
- Architecture: MVC-like, Service Locator pattern via DI.

### Ключові особливості реалізації:

#### 1. Архітектура та DI (Zenject)
- SceneContext & Installers: Використано для розділення залежностей.
- GameFactory: Реалізовано патерн *Factory* для динамічного створення гравця та автомобілів на старті сцени.
- Services: Відокремлений сервіс вводу `InputService` (реалізує `IInputService`), що дозволяє легко замінити управління (наприклад, на мобільне) без зміни логіки контролерів.
- Interfaces: Використання `IControllable` для абстрагування сутностей, якими можна керувати (людина/авто).

#### 2. Асинхронність
* Реалізовано асинхронний запуск двигуна (`async/await`, `Task.Delay`).
* Логіка руху блокується до повного завершення аудіо-послідовності (двері-стартер-двигун).

#### 3. Фізика та Управління
- WheelCollider: Налаштована крива тертя для реалістичного дрифту.
- Visuals: Синхронізація мешів коліс з колайдерами.
- Interaction: Система перемикання контексту управління (Player-Car) зі збереженням стану камери.

#### 4. Візуальні ефекти
- Post-Processing: ACES Tonemapping, Bloom, Vignette, Chromatic Aberration.
- Particles:
  - Дим з вихлопної труби (змінюється від обертів).
  - Сліди від шин (TrailRenderer) та дим при дрифті/гальмуванні.
- Audio:
    - Звуки кроків, дверей, шин.

## Управління
- WASD: Рух (персонаж та авто)
- Shift: Прискорення гравця
- Space: Гальма (авто)
- E: Сісти / Вийти з авто
- Mouse: Огляд камерою
